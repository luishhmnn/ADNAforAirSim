How to download AirSim v1.5.0 - Windows - CityEnviron:
1. Go to: https://github.com/Microsoft/AirSim/releases
2. Scroll down to release v1.5.0 - Windows (19 May 2021)
3. Read it
4. At the end of the article, click on Assets
5. Download CityEnviron.zip.001, CityEnviron.zip.002, CityEnviron.zip.003
6. 'CityEnviron, ... are big environments, so they are divided into multiple files. After downloading the 001, 002, and 003 files, 
    please use software such as 7zip to right-click on 001 file and chose one of the extract options. The 7zip automatically will 
    detect and use all files.'
7. Place directory WindowsNoEditor in this location
