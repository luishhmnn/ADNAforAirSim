# ADNAforAirSim
ADNAforAirSim is the client that enables you to control a car in AirSim using artificial DNA (ADNA).
It was developed as part of the master's thesis "Ausfallsichere Automatisierung von Kraftfahrzeugen mittels künstlicher DNA".

The scripts for executing and analysing the experiments as well as the results can be found
in repository [ADNAforAirSim_analysis](https://github.com/luishhmnn/ADNAforAirSim_analysis).
