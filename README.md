# ADNAforAirSim
ADNAforAirSim is the client that enables you to control a car in [AirSim](https://github.com/microsoft/AirSim) using artificial DNA (ADNA).
It was developed as part of the master's thesis ["Ausfallsichere Automatisierung von Kraftfahrzeugen mittels künstlicher DNA"](report/Masterarbeit_LuisHohmann.pdf).

The scripts for executing and analysing the experiments can be found
in repository [ADNAforAirSim_analysis](https://github.com/luishhmnn/ADNAforAirSim_analysis).
