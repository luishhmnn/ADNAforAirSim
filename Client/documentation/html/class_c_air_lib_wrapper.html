void CAirLibWrapper::SetSteering ( float  steering)

Sets the steering of the car.

Sets the steering of the car. Valid position range is from: 1 (left) ... 2 (middle) ... 3 (right). All other positions are invalid and ignored. This allows us to ignore 0 output when the processor with the steering actor fails.

Parameters
steeringValue between 1 and 3.
