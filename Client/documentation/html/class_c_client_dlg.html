◆ GetAcc()

float CClientDlg::GetAcc ( )

Returns the status of the ACC button.

Returns the status of the ACC button, which works as switch. If the button was clicked to activate ACC (status closed), the function returns 1.0f. If the button was clicked to deactivate ACC (status open), the function returns 0.0f.

Returns
1.0f if button's status is closed, 0.0f otherwise.

◆ SetAcc()

void CClientDlg::SetAcc ( float  value)

Sets the icon of the button to bright or dark.

Sets the icon of the ACC button to bright or dark. If 0.0f is received, the switch (m_ACCactivate) is also set to FALSE. This is necessary so that ACC does not reactivate until the user presses the button.

Parameters
value0.0f to set icon to dark, 1.0f for bright.
